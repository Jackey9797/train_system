### 安装步骤

1. 使用以下命令从GitHub克隆仓库：

    ```bash
    git clone https://github.com/yourusername/yourproject.git
    ```

2. 进入项目目录：

    ```bash
    cd train_system
    ```

3. 安装依赖：

    ```bash
    npm install
    ```

4. 启动应用程序：

    ```bash
    python manage.py runserver --insecure 0.0.0.0:8000
    ```
